# Lopez Gilbert Web Interface

Este proyecto presenta una interfaz web moderna, responsiva y optimizada para destacar el perfil profesional de Lopez Gilbert. Desarrollado con HTML, CSS y recursos visuales personalizados, el sitio combina diseño elegante con estructura técnica clara. Está desplegado públicamente mediante GitHub Pages para revisión, presentación y evaluación profesional.

## Características

- Diseño minimalista y responsivo
- Codificación limpia y semántica en HTML/CSS
- Recursos visuales alojados localmente
- Compatible con navegadores modernos
